#  Proyecto Sprint 7

Comprobación del campo name en la solicitud de creación de un kit de productos de la aplicación Urban Groceries.


## ¿De qué va el proyecto?
<p>
En este proyecto se crearon las pruebas necesarias para validar el campo name dentro de la creación de un kit de productos de la app Urban Groceries. 
Las pruebas incluyen:
</p>

- La creación de usurario, 
- La creación de un kit estableciendo únicamente el nomre de este.

**Los archivos dentro del proyecto**
1. configuration.py
2. create_kit_name_kit.py
3. data.py
4. sender_stand_request.py
5. README.mk
6. .gitignore

### Pasos para ejecutar las pruebas

##### 1. Revisar los datos

Revisar que en el archivo data.py esten definidos correctamente: 
- create_user_body
- header
- kit_name
- kit_body 1-9

##### 2. Ejectuar las pruebas con Pytest

El código ha sido escrito de manera que este se pueda ejecutar con la herramienta de automatización Pytest. Al ejecutarlo, este debe devolver cinco pruebas positivas y cuatro negativas.
